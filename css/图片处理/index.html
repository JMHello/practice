基于background - 没有使用img标签

方案一：固定高度 + background-size: cover/contain;

特点：图片定高，缩小浏览器时，图片仍会居中显示

方案二：height: 25.3125vw [+ calc()]

换算：1920/486 = 100/x （满屏图 - 100 为 宽度 100vw）

特点：缩小浏览器时，图片按比例缩小

方案三：padding - 设置百分比

特点：缩小浏览器时，图片按比例缩小

padding设置百分比 - 图片宽度自适应 - 使用img标签

padding-bottom的值为图片的高宽比 ---》 换算：1920/486 = 0.253125
